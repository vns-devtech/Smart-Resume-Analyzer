# 📖 About

Smart Resume Analyzer is a modern web application that analyzes resumes according to ATS (Applicant Tracking System) standards.

The application extracts resume content, calculates Resume Score, ATS Score, identifies missing skills, generates AI-powered suggestions, and allows users to download a professional PDF report.

---

# ✨ Features

- 📄 Resume Parsing
- 🤖 AI Resume Analysis
- 📊 Resume Score
- 🎯 ATS Compatibility Score
- 🟢 Matched Skills Detection
- 🔴 Missing Skills Detection
- 💡 AI Feedback & Suggestions
- 📑 Professional PDF Report
- 🌐 Modern Responsive UI

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/vns-devtech/Smart-Resume-Analyzer.git
````

Move into project

```bash
cd Smart-Resume-Analyzer
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Flask

```bash
python app.py
```
